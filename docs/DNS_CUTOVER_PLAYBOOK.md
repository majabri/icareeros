## Step 2 — Update Cloudflare DNS

Login: https://dash.cloudflare.com → Select `icareeros.com` zone → DNS

### Records to add/update

| Type | Name | Value | TTL | Proxy |
|---|---|---|---|---|
| `CNAME` | `@` (root) | `cname.vercel-dns.com` | Auto | **OFF (DNS only)** |
| `CNAME` | `www` | `cname.vercel-dns.com` | Auto | **OFF (DNS only)** |

> ⚠️ **Proxy must be OFF** (grey cloud, not orange). Vercel handles TLS. Orange proxy breaks Vercel's certificate provisioning.

### Records to remove/disable (old azjobs stack)

| Type | Name | Notes |
|---|---|---|
| Old `A` or `CNAME` record for `@` | Remove or replace with above |
| Old `A` or `CNAME` record for `www` | Remove or replace with above |

## Step 3 — Verify propagation

```bash
# From your terminal — check A record resolves to Vercel
dig icareeros.com +short

# Check www
dig www.icareeros.com +short

# Verify TLS + response
curl -I https://icareeros.com
# Expected: HTTP/2 200, x-vercel-id header present
```

Global propagation: usually < 5 min on Cloudflare. Full global: up to 10 min.
